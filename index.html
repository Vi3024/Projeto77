Visual Studio Code

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS.
Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Fonte: Wikipedia

Console

Console é uma tela onde você exibe sua saída e é usada para fins de teste. Antes de imprimir qualquer coisa na página HTML, é aconselhável imprimir na tela do console.

Como abrir o console?

Para abrir o console em seu computador, clique no botão direito de seu mouse dentro da página web, irá aparecer diversas opções para você clicar.
Clique na função Inspecionar, e dentro dela, clique na aba de menu console. Além do console, dentro do Inspecionar, vc também terá acesso ao código de diversas páginas web. :)
